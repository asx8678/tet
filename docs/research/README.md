# Research documentation

This directory contains documentation-only research and traceability artifacts for the Elixir CLI Agent planning/backlog work.

- [BD-0001 source inventory and missing-input report](BD-0001_SOURCE_INVENTORY_AND_MISSING_INPUT_REPORT.md) — verified source anchors, missing inputs, and research evidence surface.
- [BD-0003 phase/backlog traceability matrix](BD-0003_PHASE_BACKLOG_TRACEABILITY_MATRIX.md) — maps Phase 0 through Phase 20 to source anchors, ADRs, backlog IDs, gates, verification plans, risks, and completion evidence.

These files are planning artifacts only. They do not add application source, generated scaffolding, migrations, runtime configuration, or release files. Tiny but important distinction, because otherwise the docs gremlins start building apps when nobody asked.
